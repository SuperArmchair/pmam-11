Портфоліо реалізованих проєктів

1 Проєкт на С# – Розробка веб-платформи з інтерактивними міні-іграми та навчальними гайдами, в якій реалізовано систему реєстрації та авторизації користувачів, а для надійного зберігання даних використано СКБД PostgreSQL.

2. Проєкт на Go – Консольна симуляція екосистеми в реальному часі, де поведінка кожного організму (рослини, травоїдного, хижака) реалізована через окремі Go routines з використанням Mutexes для синхронізації даних.

3. Проєкт на Java – Розробка десктопного графічного редактора на базі бібліотеки Swing, що надає інструментарій для геометричних трансформацій (обертання) та програмної корекції кольору зображень за допомогою Java 2D API.

4. Проєкт на Python – Розробка аналітичного веб-застосунку на базі Flask, що поєднує інструменти Pandas для глибинного аналізу транспортних даних та Google Maps API для інтерактивної візуалізації трафіку та побудови оптимальних маршрутів руху.

5. Проєкт на Ruby – Розробка автономного інструментарію для організації workflow, побудованого на принципах об'єктно-орієнтованого дизайну, із вбудованим механізмом серіалізації даних та статистичного прогнозування.

6. Проєкт на Swift – Розробка системи процедурної генерації музики, що використовує ланцюги Маркова для створення мелодій та реалізує низькорівневий запис бінарного формату MIDI для експорту багатоканальних композицій.

7. Проєкт на TypeScript – Розробка консольної RPG-гри з покроковою бойовою системою, в якій реалізовано механіки прокачки персонажа, економіку ігрового магазину та поліморфну систему інвентарю на базі класів.
